## 🚀 How to Use:

### Step 1: Import Collection
```
1. Open Postman
2. Click "Import"
3. Select "SmartEduConnect_API.postman_collection.json"
4. Collection will be imported with all 180+ endpoints
```

### Step 2: Verify Base URL
```
Collection → Variables → base_url
Should be: https://smarteduconnect.in:8080/api
```

### Step 3: Login to Get Token
```
1. Go to "Authentication" → "Login"
2. Update email and password in request body
3. Send request
4. Token is automatically saved to {{token}} variable
5. All subsequent requests will use this token
```

### Step 4: Test Any Endpoint
All endpoints are organized by feature and ready to use!
